# Report

## Objective

テーマ「自己と鏡像」を、哲学・精神分析・発達心理・文学/美学の浅い root 検索で整理し、次に深掘りすべき論点を見つける。

## Distilled Findings

- 哲学の基準線では、自己意識は「自分を自分として」捉える働きであり、単に像として自分を見ることとは区別される `(N00/CQ01/A01)`。
- 現象学では、鏡像認識より前に、経験を生きている当人に伴う前反省的な自己経験がある。したがって「自己=鏡像」という還元はできない `(N00/CQ02/A02)`。
- ラカンでは、鏡像は断片的な身体感覚に対して統一的な自己像を与えるが、その統一は外部像への同一化でもあるため、自己形成は同時に疎外を含む `(N01/CQ03/A03)`。
- 発達心理の系譜では、鏡像自己認識は乳児の鏡反応や mark test で扱われるが、近年の議論ではそれは全面的な内省ではなく、主に身体的・客体的な自己把握を示すと整理される `(N02/CQ05/A05, N02/CQ06/A06)`。
- 文学と美学では、鏡は固定記号ではなく、表象・理想像・断片化・ジェンダー化された主体をめぐる動的なメタファーとして機能する `(N03/CQ07/A07, N03/CQ08/A08)`。
- ウィニコットを入れると、最初の鏡はガラスではなく養育者の顔になる。ここでは自己は視覚像だけでなく、応答され反映される関係のなかで立ち上がる `(N04/CQ09/A09)`。

## Provisional Items

- ラカンとウィニコットを並べると、「鏡」は自己をまとめる装置であると同時に、他者依存や疎外を露出する比較軸になる。ただしこの橋渡し自体は二次文献に依存するので、次段では一次文献どうしの精査が必要 `(N04/CQ10/A10)`。
